# 特定商取引法に基づく表記(ドラフト)

※このドキュメントはドラフトです。特定商取引法上、有料販売を行う場合はこの表記が法律上必須です。
　個人事業主の場合の記載範囲(住所・電話番号の扱い等)は近年ルールが見直されているため、
　公開前に必ず最新の消費者庁のガイドライン、または専門家(行政書士・弁護士等)にご確認ください。
　〔　〕内は要記入・要確認箇所です。

---

| 項目 | 内容 |
|---|---|
| 販売事業者名 | 〔氏名(屋号がある場合は屋号も)〕 |
| 運営統括責任者 | 〔氏名〕 |
| 所在地 | 〔住所〕※個人事業主の場合、請求があれば遅滞なく開示する方法での代替が認められる場合があります。最新ルールを要確認 |
| 連絡先 | 〔電話番号・メールアドレス〕 |
| 受付時間 | 〔例:メールでのお問い合わせは常時受付、回答は3営業日以内〕 |
| 販売価格 | 単発プラン:¥480(税込)/ 月額プラン:¥480/月(税込) |
